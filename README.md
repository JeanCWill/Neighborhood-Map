# Neighborhood-Map

Projeto do mapa da vizinhança para o Nanodegree de Front-End da Udacity

## Intalação

Para instalar e rodar o projeto é necessário:
- Realizar o download ou clone do projeto no [GitHub](https://github.com/JeanCWill/Neighborhood-Map.git)
- Possuir um usuário na api do [Foursquare](https://developer.foursquare.com/)
- Adicionar seu **CLIENT_ID** na variável `clientIdFoursquare` dentro do arquivo **app.js**
- Adicionar seu **CLIENT_SECRET** na variável `clientSecretFoursquare` dentro do arquivo **app.js**
- Possuir uma chave de acesso do Google Maps API
- Adicionar sua **API_KEY** na url da api do Google Maps no **index.html** substituindo o texto `????`
- Executar o arquivo **index.html**

## API's utilizadas
- [Foursquare](https://developer.foursquare.com/)
- [Maps](https://developers.google.com/maps/?hl=pt-br)
